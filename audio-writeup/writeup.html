Overivew and Usage

My project consists of 6 various sketches created in p5 that encompass various attributes. Some of these attributes are a static and random image, an interactive design, as well as an audio and audio/input drawing.

My gifs and other images are presented at the top

To run this code you can go to my github repo which is listed below and clone my audio-writeup repo. In that repo you can find a folder titled sketches in which you will find the code to each of my sketches. You can copy this code and run it in p5.

Development Process

My step by step process consisted of:

  1. Looking at various examples done in p5
  2. Starting off with a concept of what I wanted to build and update across the sketches
  3. Coding each sketch and then changing it based on what I thought would look better
  4. Looking up differnet ways to incorporate audio/randomness/etc.
  5. Add researched code into p5 sketches

Issue Deep-Dive

One of the problems I encountered was adding in audio for my 5th sketch as well as responding to audio input for the 6th sketch. Specifically for the 5th sketch I wasn't able to figure out how to get audio to appear in the sketch. I was able resolve this through watching the coding train videos that were offered as well as looking at some of his other videos on Youtube. I also watched some other implementations of audio into p5 that helped me. Similarly with figuring out how to create a sketch that took in audio input, I looked at the p5 examples in relation to this topic as well as saw some other Youtube videos.

Another problem I encountered was with the grid layout and image sizing. I really struggled to get the grid set up nicely on my portfolio webpage. I was able to resolve this through looking at Hannah's code examples on github as well as go to office hours and get some assistance from Raina with setting up a flexbox.
