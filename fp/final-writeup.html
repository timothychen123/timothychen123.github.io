Final Project Writeup


Drawing Game

Overivew and Usage

My project is a drawing game similar to that of pictionary. For those that may not know this game, it involves choosing how much time you would like to draw the a randomly generated word and then drawing that word in another screen.

Here is a gif of my game.

To run this code you can go to my github repo which is listed below and clone it. You can copy the code in the index.js, index.html, and popup.html and put it into p5 and play the game or run it locally as well.

Development Process

My step by step process consisted of:

  1. Taking in feedback from the showcase in 438 class
  2. Ideating on the implementation of the features such as multiple colors and choosing a time option
  3. Adding more words into the word bank
  4. Adding in the colors into p5 and checking to see if it worked
  5. Adding in different size pen sizes to use when drawing
  6. Adding in a popup to choose time options
  7. Fixing issues with selecting a time option and having it connect to the index.js file

Issue Deep-Dive

One of the problems I encountered was in adding colors and then having the color be given to the pen. One of the main issues I was having was getting the correct x and y coordinates of the ellipse of a specific color and then reflecting that to the pen. I was able to solve this through trial and error where I would put in an x and y coordinate I thought would work then tested it and then adjusted if it didn't work. Through this process I was eventually able to get the correct colors to correspond when a user clicks on a specific color.

Another problem I encountered was that in my popup.html, when the user selected a time it wasn't saving that variable and allowing for me to use it in index.js for my drawing game. I was able to get a lot of support from Raina in which she directed me to session storage which we were able to implement and that solved the issue I was having. By saving my variable to the session I could use it in my index.js file.
